# MK Home Assistant Collection

This repository contains my custom cards and components for Home Assistant.

## Installation via HACS

1.  Open **HACS** in your Home Assistant instance.
2.  Go to **Frontend**.
3.  Click the 3 dots in the top right -> **Custom repositories**.
4.  Add the URL of this repository:
    `https://github.com/mcknight89/mk.ha`
5.  Select Category: **Plugin**.
6.  Click **Add**.
7.  Search for "MK Home Assistant Collection" and install.

## Included Components

- **MK Network Device Card**: A custom card for managing network devices.
- **MK State Card**: A custom visualization for entity states.
